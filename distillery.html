Distillery

Distillery helps you decide what to play with your Steam friends by showing you which games you have in common.

Friends

The friends of the user you input above will be displayed here. Clicking a friend will add that friend to Selected Users. Users with an orange or red border aren't visible due to their privacy settings - you won't be able to see which games they own.

Selected Users

Users who appear here will have their owned games shown below. To select a user, you can input a user's custom URL in the box above if they've chosen one: http://steamcommunity.com/id/myCustomURL Or their ID number, as seen in their default profile page URL: http://steamcommunity.com/profiles/12345678901234567 Clicking a user here will deselect that user.

Games

Games owned by the selected users above will be listed here. The games will be sorted by the number of selected users who own them - that is, the games which the most users have in common will be at the top. You can see which users own or don't own a particular game by clicking on the number of owners.
